## Registers

x86_64 exposes sixteen 64-bit general-purpose registers. Their lower halves
are addressable as 32-, 16- or 8-bit registers.

| 64-bit | 32-bit | 16-bit | 8-bit |
|:-------|:-------|:-------|:------|
| rax | eax | ax | al |
| rbx | ebx | bx | bl |
| rcx | ecx | cx | cl |
| rdx | edx | dx | dl |
| rsi | esi | si | sil |
| rdi | edi | di | dil |
| rbp | ebp | bp | bpl |
| rsp | esp | sp | spl |
| r8..r15 | r8d..r15d | r8w..r15w | r8b..r15b |

### Caller-saved (scratch)
`rax, rcx, rdx, rsi, rdi, r8, r9, r10, r11`

### Callee-saved (preserved)
`rbx, rbp, rsp, r12, r13, r14, r15`

`rax` carries the return value (when ≤ 64 bits). Larger return types use the
stack per the ABI.
